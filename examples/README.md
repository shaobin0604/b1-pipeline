# Example Artifacts

This directory contains **sanitized examples** that show what a daily B1 pipeline run may produce.

These files are intentionally small, fake/sanitized, and suitable for repository display.

## Included examples

- `sample_review_result.json` — a simplified review output example
- `sample_summary.md` — a simplified final review summary example

## Notes

- These examples are not real trading recommendations.
- Values are illustrative only.
- Real local run outputs are normally generated under `data/` and are not committed.
